# Formulario de Reclamación de Garantías

Este proyecto es una aplicación web de una sola página (SPA) diseñada para simplificar y estandarizar el proceso de reclamación de garantías. Fue desarrollada como una solución a medida para **CV Tools**, con el objetivo de proporcionar a sus clientes una herramienta intuitiva y eficiente para reportar incidencias de productos.
